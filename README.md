# Microsoft Entra ID Identity & Access Management Security Lab
 
## Overview
 
This project demonstrates hands-on experience with Microsoft Entra ID administration, Identity and Access Management (IAM), Role-Based Access Control (RBAC), and cloud security fundamentals.
 
## Environment
 
- Microsoft Azure
- Microsoft Entra ID
- Microsoft 365 Admin Center
 
## Objectives
 
- Build a Microsoft Entra ID tenant
- Manage identities and users
- Implement Role-Based Access Control (RBAC)
- Review Microsoft security recommendations
- Analyze Secure Score recommendations
- Explore authentication and password policies
- Review audit and sign-in logs
 
## Activities Completed
 
### Tenant Administration
 
- Created and managed a Microsoft Entra ID tenant
- Managed Azure subscription integration
 
### User Management
 
- Created Employee accounts
- Created HelpDesk user account
- Created Break Glass Administrator account
 
### RBAC Implementation
 
- Assigned Helpdesk Administrator role
- Assigned Global Administrator role
- Reviewed built-in Microsoft Entra roles
 
### Security Review
 
- Reviewed Identity Secure Score
- Validated Password Expiration Policy
- Reviewed Authentication Methods
- Explored Sign-in Logs
- Explored Audit Logs
 
### Security Best Practices Applied
 
- Administrative redundancy
- Least privilege principles
- Break Glass account implementation
- Password policy validation
 
## Skills Demonstrated
 
- Microsoft Entra ID
- Azure Administration
- Identity and Access Management (IAM)
- RBAC
- Microsoft 365 Administration
- Security Administration
- Cloud Security Fundamentals
